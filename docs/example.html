Getting Started

  • Clone the repository from GitHub: git clone https://github.com/our-project.git
  • Install the dependencies using npm: npm install
  • Start the development server: npm start

Project Structure

  • src: Contains the source code for the project

  • tests: Contains the unit tests and integration tests
